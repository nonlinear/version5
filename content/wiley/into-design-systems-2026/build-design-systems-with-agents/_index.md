# Building real design systems with agents

---

### Colocation
Component code, tests, docs, and examples - all in one directory
When an agent looks for a component, everything it needs is right there
No cross-referencing across repos, wikis, and Storybook in multiple locations
One folder = one complete truth

---

### Naming
Names that tell an agent what to use and when green-600 is a guessing game
Semantic names encode intent — arbitrary names encode nothing
The name IS the documentation
